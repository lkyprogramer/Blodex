---

## 4. 技术方案

### 4.1 Nightmare 调优

1. 先锁定当前 Nightmare 超标来源：
   - floor pacing
   - skill cadence
   - sustain
   - spike frequency
2. 只允许针对 Nightmare 或高难分支做局部调优。
3. 调优后必须重新生成 evidence pack。

### 4.2 三职业 parity

每个职业至少一局完整 run，记录：

1. 开局技能池
2. 核心选择密度
3. 生存压力
4. build identity 成型速度

### 4.3 buff / damageType 合同验证

必须至少覆盖：

1. `war_cry`
2. `shadow_step`
3. `frost_nova`
4. `physical` vs `arcane` 对 armored 目标

### 4.4 最终签署

1. 更新 regression matrix
2. 更新 taste sign-off
3. 更新 release readiness
4. 补齐录像链接或归档路径

### 4.5 资源策略

1. 本阶段以证据补齐为主，不以资源新增为主。
2. 如 Nightmare 调优后需要对现有反馈强度做微调，优先复用：
   - 现有 SFX key
   - 现有 VFX 参数
   - 现有截图/录像归档模板
